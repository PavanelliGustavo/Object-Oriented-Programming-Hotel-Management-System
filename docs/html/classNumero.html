Detailed Description

Domínio para representar o Número do Quarto em um Hotel.

Esta classe armazena um valor inteiro que representa o número do quarto. O formato válido, de acordo com as regras de negócio, deve ser um valor entre 1 e 999 (001 a 999). A classe garante a validade do seu valor interno através do método validar.
