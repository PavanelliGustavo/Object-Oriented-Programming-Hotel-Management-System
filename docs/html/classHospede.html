Detailed Description

Entidade que representa um cliente (Hóspede) do hotel.

Herda de Pessoa e adiciona atributos necessários para o processo de reserva e faturamento.

Atributos adicionais:

  • Endereço: Localização residencial do hóspede.
  • Cartão: Dados de pagamento.
