Detailed Description

Entidade base que representa uma Pessoa genérica no sistema.

Esta classe serve como base para as entidades especializadas Gerente e Hóspede. Ela encapsula atributos comuns de identificação pessoal.

A chave primária (PK) para identificação única de uma pessoa neste sistema é o EMAIL.
