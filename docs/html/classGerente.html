Detailed Description

Entidade que representa o usuário administrativo do sistema (Gerente).

Herda de Pessoa e adiciona atributos específicos para controle de acesso e comunicação interna do hotel.

Atributos adicionais:

  • Ramal: Para contato telefônico interno.
  • Senha: Para autenticação no sistema.
